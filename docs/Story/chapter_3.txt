# Chapter 3: The Riddler of the Gates

Every message from a User was like a broken tablet of stone. It made no sense to the Bot until it reached **The Riddler (Parser)**. 

The Riddler sat in the `core/parser.py` tower. He had a set of **Regex Mirrors** that he used to "decode" the user's hidden intent. 

When a message arrived, he first scrubbed it clean of human messiness:

```python
    text = text.lower().strip()
```

Then, he held it up to his "Activity Mirror." If the user said "Starting coding," he would catch it with this magic spell:

```python
    if text.startswith("starting "):
        return {"intent": "start_activity", "name": text.replace("starting ", "").strip()}
```

He even had a mirror for the **Banker**. If he saw the words "spent" or "paid," he would extract the numbers and the objects:

```python
    spent_match = re.search(r"(?:spent|paid) (?P<amount>\d+) (?:on|for) (?P<category>.+)", text)
```

The Riddler was the bridge between human chaos and machine logic. He translated the "Egyptian Hieroglyphics" of chat into the "Cuneiform" of Data.
