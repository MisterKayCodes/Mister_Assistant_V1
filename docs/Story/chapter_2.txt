# Chapter 2: The Watchman and The Inspector

Beyond the heart stood **The Bootstrapper (run.py)**. He was the one who actually built the world before Main could wake up. 

He used a specialist called **The Inspector (architecture_inspector.py)**. The Inspector had a list of the "Organism's" vital organs. He would scan the directory to make sure every folder (`core`, `bot`, `data`) was in the right place:

```python
    if scan_organism():
        # The Inspector says we are healthy!
        ...
```

If the Inspector found a missing limb, he would scream for **The Debugger**. But if the body was whole, the Bootstrapper summoned **The Watchman (watchfiles)**.

The Watchman was dangerous. He would restart the entire world if he saw a single change. To keep things stable, the Architect gave him **The Shield (WatcherShield)**:

```python
    class WatcherShield(DefaultFilter):
        def __call__(self, change, path):
            if "data" in path or path.endswith((".db", ".sqlite", ".log", ".git", ".tmp")):
                return False # IGNORE the noise!
            return super().__call__(change, path)
```

With his Shield, the Watchman became the Bot's protector. He would only restart the heart if the Code changed, allowing the Bot's memory (the database) to grow in silence:

```python
    run_process(base_dir, target=f"{sys.executable} main.py", watch_filter=WatcherShield())
```
