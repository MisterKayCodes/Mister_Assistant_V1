# Chapter 4: The Diplomatic Strategist

Once the Riddler has done his job, he has a label like "Log_Spending" or "Start_Activity." But the machine's internal records are cold and ugly. 

Enter **The Strategist (Logic.py)**. He is the master of the "User Experience." He takes the cold facts and wraps them in a warm 🤖 response.

When the Bot successfully starts a task, the Strategist chooses a **Scroll of Confirmation**:

```python
    def format_start_message(self, name):
        return f"⏱️ Tracking {name}"
```

If the Bot switch activities, the Strategist performs a "Summation" ritual, calculating the minutes before starting the next task:

```python
    def format_switch_message(self, prev_duration_seconds, next_name):
        res = ""
        if prev_duration_seconds is not None:
            mins = prev_duration_seconds // 60
            res = f"✅ Previous activity was {mins}m. "
        return res + f"⏱️ Now tracking {next_name}"
```

He is the "Voice" of the Assistant. He ensures that every interaction feels polished and professional. He even has a "Safety Scroll" for when things go wrong:

```python
    def format_unknown(self):
        return "🤖 I'm not sure how to handle that yet."
```
