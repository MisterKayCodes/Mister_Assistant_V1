# Chapter 5: The Immortal Archivist (The Vault)

Deep beneath the City lives **The Archivist (Repository)**. He is the one who remembers everything. He manages the **Vault of SQLite**.

In the old world, the Archivist was solitary. If he was writing, no one else could even look at the vault. But the Architect rebuilt him with the **Revolving Door (WAL Mode)**:

```python
    self.conn = sqlite3.connect(DB_PATH, timeout=20, check_same_thread=False)
    # Enable WAL mode for better concurrency
    self.conn.execute("PRAGMA journal_mode=WAL")
```

The Archivist's first job is the **Creation of the Scrolls** (The Tables). He carves the structure of the world into stone:

```python
        cursor.execute("""
            CREATE TABLE IF NOT EXISTS activities (
                id INTEGER PRIMARY KEY AUTOINCREMENT,
                user_id TEXT,
                name TEXT,
                start_time TIMESTAMP,
                ...
            )
        """)
```

He uses the **"Atomic Promise" (conn.commit)** to ensure that if he starts writing, he finishes correctly. Every connection is a single, eternal link (the `self.conn` instance) that never dies as long as the Bot is awake.
